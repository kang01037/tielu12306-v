# 铁路12306售票系统接口文档

## 1. 概述

本文档描述铁路12306售票系统各微服务的RESTful API接口。

### 1.1 技术架构

本系统采用 **Spring Cloud Alibaba** 微服务架构，使用 **Nacos** 作为服务注册与发现中心，各服务间通过 **OpenFeign** 进行HTTP通信。

### 1.2 服务列表

| 服务名称 | 端口 | 说明 |
|---------|------|------|
| service-user | 8081 | 用户服务 |
| service-train | 8082 | 列车服务 |
| service-ticket | 8083 | 余票服务 |
| service-order | 8084 | 订单服务 |
| service-pay | 8085 | 支付服务 |
| gateway | 8080 | API网关 |

### 1.3 服务调用关系

```
前端 → 网关(8080) → 各业务服务
                   │
                   └── service-order 调用 ──→ service-user (Feign)
                                    ──→ service-train (Feign)
                                    ──→ service-ticket (Feign)
```

**基础URL**: `http://localhost:8080` (通过网关)

**通用响应格式**:
```json
{
  "code": 200,
  "message": "success",
  "data": {},
  "timestamp": 1716259200000
}
```

**响应码说明**:
| 响应码 | 说明 |
|--------|------|
| 200 | 成功 |
| 400 | 参数错误 |
| 401 | 未授权 |
| 403 | 禁止访问 |
| 404 | 资源不存在 |
| 500 | 系统异常 |
